# Email Server Alternatives

## If mail.bpayapp.co.ke doesn't work, try:

### Option 1: Host Africa SMTP
```
EMAIL_HOST=smtp.hostafrica.co.za
EMAIL_PORT=587
```

### Option 2: Host Africa Mail Server
```
EMAIL_HOST=mail.hostafrica.co.za
EMAIL_PORT=587
```

### Option 3: Use Gmail SMTP (Temporary)
```
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your-gmail@gmail.com
EMAIL_PASS=your-app-password
```
